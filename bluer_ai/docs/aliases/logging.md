# logging

```bash
@log \
	<message>
 . log message.
@error \
	<message>
 . error <message>.
@warn \
	<message>
 . warn <message>.
@log::list \
	<this,that> \
	[--before "list of"] \
	[--after "items(s)"] \
	[--delim space | <delim>]
 . log list.
@log \
	verbose \
	[on | off]
 . verbose logging on/off.
```
